# Backend приложения для записи на приём к врачу

Учебный проект для колледжа.  
Приложение реализовано на Python и работает через консоль.

## Возможности

- добавление пациентов;
- добавление врачей;
- создание расписания врачей;
- запись пациента на приём;
- просмотр всех записей;
- отмена записи.
